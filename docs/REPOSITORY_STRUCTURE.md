# QNHD-Fold Repository Structure

## Directory Layout

```
QNHD-Fold/
├── README.md
├── LICENSE
├── requirements.txt
├── setup.py
├── qnhd_fold/          # Main package
├── benchmarks/         # Evaluation scripts
├── examples/           # Usage examples
├── paper/              # Publication materials
├── docs/               # Documentation
└── tests/              # Unit tests
```

## Key Components

- **qnhd_fold/**: Core implementation
- **benchmarks/**: CASP15, membrane proteins, IDRs
- **paper/**: LaTeX manuscript and figures
- **examples/**: Quick start scripts
